Commands
The commands are not case sensitive
--Everyone--
Help - Lists available commands
Isgay [@user] - Finds out if user is gay
Author - Tells you who made the bot
Website - Gives you the website url
Name-is? - Tells you a bit about the bot
Injurycounter - tells you the time since your last injury
Resetinjurycounter - Resets the time since your last injury
And informs you how long ago it was
--Admins--
Kick [@user] [reason] - Kicks the mentioned user
Ban [@user] [reason] - Bans the mentioned user
Serverprefix - Changes this servers command prefix
Bulkdel [amount] - Deletes as many messages as you want
Setlevelchannel - Sets the LevelUp channel to the current channel
Shouldping - Sets whether or not the levelling user is pinged (1/0)
--Owner--
Isplaying - Sets status for next restart
Restart - Restarts the bot faster than you can blink
Updatestatus - Loads game from FireBase
ResetServer - [id] [prefix] Overwrites website data
NewGame - [text] Sets game manually, without overwriting FireBase
